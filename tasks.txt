Задания:
*Сделать разметку в соответствии с макетом: h1, img, input, button, ul, button. 
Это все может быть обернуто в section, для удобной записи стилей. Input и button 
желательно обернуть в тег div. Стили (цвета, размер шрифта и др.) добавлять по желанию,
можно воспользоваться стилями выше.
1. При нажатии на картинку изменить изображение на другое.
2. При вводе текст в input и последующем нажатии на кнопку "Добавить", должен 
быть создан тег li (с текстом из input) и добавлен в ul. После отображения значения
в списке, текста в input быть не должно.
3. При введении более 5 дел, сделать кнопку "Добавить" заблокированной (disabled).
4. Если в input нет никакого значения, то сделать рамку input красного цвета, а фон - 
розового.
5. Если нажать на одно из записанных дел, оно должно быть перечеркнутым. Использовать
element.classList.toggle('className').
6. При фокусе сделать рамку синего цвета (focus), а фон (backgroundColor) - светло-голубого 
цвета. При потере фокуса (blur) - серого цвета, а фон - светло-серого цвета.
7. При нажатии на кнопку "Удалить все", удалить все созданные ранее пункты списка.
